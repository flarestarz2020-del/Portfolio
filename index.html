About

Bit of background

I'm based in Finland. I started building sites because I needed one for a project and didn't want to use a template. Turns out other people had the same problem, so I started doing it for them too.

I work evenings. 5 PM to 8 PM is my main window, though I can flex when something needs to move fast. I don't run a studio, I don't have a team. It's just me, which means you deal with one person start to finish.

I write everything in HTML, CSS, and JS by hand. The reason is pretty simple: hand-written code loads faster, ranks better in search, and doesn't break two years later because a plugin stopped being maintained. If you need to edit content yourself, I'll set up a lightweight CMS on top of that. The underlying code stays clean either way.
